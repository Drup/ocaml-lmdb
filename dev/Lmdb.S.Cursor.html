Module Lmdb.S.Cursor

module Cursor: sig .. end  with type db := t

Manual iterators.

type -[< `Read | `Write ] t

A cursor allows to iterates manually on the database. A cursor may be read-only or read-write.

val go : ([< `Read | `Write ] as 'b) Lmdb.cap ->
       t -> ?txn:'b Lmdb.Txn.t -> ('b t -> 'a) -> 'a

go cap db ?txn f makes a cursor in the transaction txn using the function f cursor.

The function f will receive the cursor. A cursor can only be created and used inside a transaction. The cursor inherits the permissions of the transaction. The cursor should not be leaked outside of f.

Here is an example that returns the first 5 elements of a db:

go ro db begin fun c ->
  let h = first c in
  let rec aux i =
    if i < 5 then next c :: aux (i+1)
    else []
  in
  h :: aux 1
end
txn : if not provided, a transaction will implicitely be created and committed after f returns.
val get : [< `Read | `Write ] t -> Lmdb.S.key * Lmdb.S.elt

get cursor returns the binding at the position of the cursor.

val put : [< `Read | `Write > `Write ] t ->
       ?flags:Lmdb.PutFlags.t -> Lmdb.S.key -> Lmdb.S.elt -> unit

put cursor k v adds k,v to the database and move the cursor to its position.

val put_here : [< `Read | `Write > `Write ] t ->
       ?flags:Lmdb.PutFlags.t -> Lmdb.S.key -> Lmdb.S.elt -> unit

put_here cursor k v adds k,v at the current position.

  • Raises Error if the key provided is not the current key.
val remove : [< `Read | `Write > `Write ] t -> ?all:bool -> unit -> unit

remove cursor removes the current binding.

If the database allow duplicate keys and if all is true, removes all the bindings associated to the current key.

val first : [< `Read | `Write ] t -> Lmdb.S.key * Lmdb.S.elt

first cursor moves the cursor to the first binding.

val last : [< `Read | `Write ] t -> Lmdb.S.key * Lmdb.S.elt

last cursor moves the cursor to the last binding.

val next : [< `Read | `Write ] t -> Lmdb.S.key * Lmdb.S.elt

next cursor moves the cursor to the next binding.

val prev : [< `Read | `Write ] t -> Lmdb.S.key * Lmdb.S.elt

prev cursor moves the cursor to the prev binding.

val seek : [< `Read | `Write ] t -> Lmdb.S.key -> Lmdb.S.elt

seek cursor k moves the cursor to the key k.

val seek_range : [< `Read | `Write ] t -> Lmdb.S.key -> Lmdb.S.elt

seek_range cursor k moves the cursor to the first key greater or equal to k.

Operations on duplicated keys

Similar to the previous operations, but only inside a set of binding sharing the same key.

Raise Invalid_argument if used on a database that does not support duplicate keys.
